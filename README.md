# SME-PRINTER
Repositório para controle de versão do Projeto Integrador III da Univesp do primeiro semestre de 2026, Grupo 09

## O que é?

A idéia deste projeto é utilizar um script Python para estar retirando os valores de _Contadores_ e _Níveis de Toners_ de um servidor Zabbix, salvando em um banco de dados.

## Qual o objetivo?

O objetivo ao utilizar estas informações é conseguir criar uma apresentação e análise desses dados.

## Quais tecnologias estão sendo utilizadas?
Atualmente, o repositório consta apenas com os scripts _Python_, o script _Python_ aqui é deixado em um servidor Linux que atualmente consta com o sistema Zabbix
